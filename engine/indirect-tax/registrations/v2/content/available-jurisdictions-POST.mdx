---
title: "List available jurisdictions"
api: "GET https://transaction-tax.api.in.commenda.io/api/v1/registrations/v2/content/jurisdictions"
description: "Search for jurisdictions where you can create V2 tax registrations"
---

Search for tax jurisdictions where V2 registrations are available. Returns jurisdictions that match your filters, with cursor-based pagination for large result sets.

## Query parameters

<ParamField query="corporation_id" type="string" required>
  The unique identifier for the corporation.
</ParamField>

<ParamField query="countries" type="array">
  Filter by ISO 3166-1 alpha-2 country codes.
  
  Pass multiple values as comma-separated: `countries=US,DE`
</ParamField>

<ParamField query="states" type="array">
  Filter by state or province codes.
  
  Pass multiple values as comma-separated: `states=CA,NY,TX`
</ParamField>

<ParamField query="types" type="array">
  Filter by jurisdiction type.
  
  Values: `TRADEBLOC`, `COUNTRY`, `STATE_OR_PROVINCE`, `CITY`, `COUNTY`, `DISTRICT`, `LOCAL`, `TERRITORY`
</ParamField>

<ParamField query="search" type="string">
  Free-text search across jurisdiction names.
  
  Example: `"California"` or `"Germany"`
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor from a previous response. Leave empty for the first page.
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Maximum number of results to return (1–100).
</ParamField>

```bash cURL - US states

  --header 'Authorization: Bearer <your_token>'
```
