---
title: "Bulk create registrations"
api: "POST https://transaction-tax.api.in.commenda.io/api/v1/registrations/v2/bulk"
description: "Create multiple V2 tax registrations in a single request"
---

Create multiple registrations for a corporation in one request. Each registration in the array is validated and created independently — if one fails, others may still succeed.

## Request body

<ParamField body="corporation_id" type="string" required>
  The unique identifier for the corporation.
</ParamField>

<ParamField body="registrations" type="array" required>
  Array of registration objects to create. Minimum 1 item.

  <Expandable title="Registration item fields">
    <ParamField body="registrations[].registration_tax_type_id" type="string" required>
      The content ID identifying the jurisdiction and tax type combination.
    </ParamField>

    <ParamField body="registrations[].tax_types" type="array" required>
      Array of tax types to register for. Must be valid for the selected `registration_tax_type_id`. Minimum 1 item.
    </ParamField>

    <ParamField body="registrations[].frequencies" type="array" required>
      Array of filing frequencies. Must match an allowed frequency combination. Minimum 1 item.
    </ParamField>

    <ParamField body="registrations[].effective_start_date" type="string">
      ISO date (YYYY-MM-DD) when the registration became or becomes effective.
    </ParamField>

    <ParamField body="registrations[].registered_by" type="string">
      Who is creating this registration. Values: `COMMENDA`, `API`, `USER`
    </ParamField>

    <ParamField body="registrations[].tax_registration_number" type="string">
      Tax registration ID or permit number issued by the tax authority.
    </ParamField>

    <ParamField body="registrations[].credential_id" type="string">
      Reference to stored portal credentials (UUID).
    </ParamField>

    <ParamField body="registrations[].member_state_registration_id" type="string">
      UUID of an existing active domestic registration. Required for Union OSS registrations.
    </ParamField>

    <ParamField body="registrations[].registered_through_registration_tax_type_id" type="string">
      Content ID of the member country to register through. Required for Non-Union OSS and IOSS registrations.
    </ParamField>
  </Expandable>
</ParamField>
