The **Ownership** tab lets you track who owns each entity, including individual stakeholders, parent companies, and share class details.

## Viewing ownership

1. Go to **Entities** and select an entity.
2. Click the **Ownership** tab.

Here you can see all stakeholders and their ownership percentages.

## Adding stakeholders

Click **Add stakeholder** to record a new owner. For each stakeholder, you can track:

- **Name** and contact information
- **Ownership percentage**
- **Share class** (e.g., common, preferred)
- **Number of shares**

## Share classes

If your entity has multiple classes of stock, you can define each share class with details like:

- Class name (Common, Preferred Series A, etc.)
- Number of authorized shares
- Par value
- Voting rights

## Cap table

For entities with equity transactions (issuances, transfers, etc.), Commenda maintains a cap table that tracks the history of ownership changes over time.

<Info>
The cap table feature is especially useful for startups and companies with multiple funding rounds. It provides a clear audit trail of all equity transactions.
</Info>
